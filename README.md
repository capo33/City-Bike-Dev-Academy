## City-Bike-Dev-Academy-task

## Installation

1- Clone the repo 
```bash
https://github.com/capo33/City-Bike-Dev-Academy
```
2- install dependencies 
```bash
yarn 
```
3- Create .env file
4- add your own data in .env file
```bash
PORT= your PORT
MONGODB_URI = your MONGODB URI
```

## Usage
1- Go to the root  directory and start the server
```
yarn start
```
